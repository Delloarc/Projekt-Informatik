In dieser Stunde habe ich die Navigationsleiste erstellt, mit 3 Links (Startseite, Dokumentation und Kontakt). Damit es schön aussieht, habe ich ein Bild von Pexels (urheberrechtfreie Bilder) genommen und es als background bei CSS gesetzt. Damit man den Text im Vordergrund lesen kann, habe ich den Hintergrund etwas dunkler gemacht, mit background-blend-mode: multiply; und einer background-color, die etwas grau ist. Die Links habe ich im CSS mit display: flex; angeordnet, damit diese horizontal nebeneinander stehen. Damit man sieht, auf welcher Seite man gerade ist, habe ich class="active-link" bei dem Link hinzugefügt, auf dessen Seite man ist und diese class dann im CSS mit border-bottom designt. Damit die Seite einen Rand hat, habe ich für den .content wrapper margin hinzugefügt. Da mir die Standart Schriftart nicht gefällt, habe ich die Schriftart sans-serif genommen und die im * {} universal selector für alle Texte benutzt.
